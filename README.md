Gradient Boosting From Scratch
Overview

This project implements Gradient Boosting from scratch using Python.
Instead of relying on pre-built libraries, the goal is to understand how boosting works internally — how weak learners are trained sequentially, how residual errors are calculated, and how predictions improve step by step.

The implementation focuses on clarity, learning, and core machine learning concepts.
